# 현장도우미 사용 설명서 / 现场助手 操作手册

> v1.1 — 最后更新: 2026-05-19

---

## 📱 앱 열기 / 打开应用

카카오톡 또는 휴대폰 브라우저에서 열기:

```
https://hyunjang-doumi.vercel.app   (배포 후)
http://localhost:3000               (로컬 개발)
```

---

## 1단계: 로그인 / 第 1 步：登录

### 방법 A: 간편 로그인 (추천) / 一键登录（推荐）

| 버튼 | 대상 |
|------|------|
| **Google 로그인** | Google 계정 있는 사람 (한국인 대부분) |
| **카카오 로그인** | 카카오톡 계정 있는 사람 |
| **위챗 로그인** | 위챗 계정 있는 사람 (중국인, 준비중) |

### 방법 B: 이메일 로그인 / 邮箱登录

이메일 + 비밀번호 입력 → 회원가입 또는 로그인

---

## 2단계: 프로필 설정 (최초 1회) / 第 2 步：填写个人信息（仅首次）

로그인 후 반드시 다음 정보를 입력해야 합니다:

| 필드 | 설명 | 필수 |
|------|------|------|
| **역할 (角色)** | 현장소장 / 작업자 | ✅ |
| **실명 (真实姓名)** | 실제 이름 | ✅ |
| **전화번호 (手机号)** | 010-xxxx-xxxx | 선택 |
| **공종 (工种)** | 목공/전기/설비/도배/타설/기타 | 작업자만 ✅ |

> ⚠️ 실명을 입력하지 않으면 사용할 수 없습니다. 허위 등록 방지.
> ⚠️ 不填写真实姓名无法使用系统。

---

## 3단계: 현장 등록 (소장 전용) / 第 3 步：添加工地（所长专用）

| 순서 | 화면 |
|------|------|
| 1 | 하단 "현황" 탭 → [+] 버튼 |
| 2 | 현장 이름 입력 (예: 강남 리모델링) |
| 3 | "Add Site" 클릭 |

---

## 4단계: 작업자 초대 (소장 전용) / 第 4 步：邀请工人（所长专用）

| 순서 | 화면 |
|------|------|
| 1 | 하단 "작업자" 탭 (👥 아이콘) |
| 2 | 초대할 현장 선택 |
| 3 | 작업자의 사용자 ID 입력 → "초대" 클릭 |

> 작업자가 먼저 회원가입 후 프로필 설정까지 완료해야 초대 가능합니다.
> 工人需要先注册并完成实名认证才能被邀请。

---

## 5단계: 퇴근 체크인 / 第 5 步：下班打卡

**언제?** 매일 퇴근 전 (오후 5시)  
**누가?** 모든 작업자

| 순서 | 설명 |
|------|------|
| 1 | 하단 "출퇴근" 탭 |
| 2 | 현장 선택 |
| 3 | 공종 선택 |
| 4 | 상태 선택: ✅ 완료 / 🕐 반일 연기 / 🔴 하루 연기 |
| 5 | 사진 촬영 |
| 6 | "제출" 클릭 |

---

## 6단계: 문제 신고 / 第 6 步：问题上报

**언제?** 누수, 크랙, 평활도 불량 등 문제 발견 시

| 순서 | 설명 |
|------|------|
| 1 | 하단 "보고" 탭 |
| 2 | 문제 부위 사진 촬영 |
| 3 | 사진에 동그라미 그리기 ("Mark on photo") |
| 4 | 문제 유형 선택 (누수/불평활/크랙/방수/청소/기타) |
| 5 | "길게 눌러 말하기" → 중국어로 설명 |
| 6 | "한국어로 번역" → AI 자동 번역 |
| 7 | "보고서 전송" |

## ❓ FAQ

**Q: 앱 설치가 필요한가요?**  
A: 아니요. 브라우저에서 링크만 열면 됩니다.

**Q: 외국인 작업자도 쓸 수 있나요?**  
A: 네. AI 음성 번역으로 중국어→한국어 자동 변환. 언어 UI도 중국어 지원.

**Q: 사진은 어디에 저장되나요?**  
A: AWS 서울 리전에 안전하게 저장됩니다.
