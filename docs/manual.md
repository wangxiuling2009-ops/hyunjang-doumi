## 1단계: 회원가입 / 第 1 步：注册

| 순서 | 화면 | 설명 |
|------|------|------|
| 1 | 로그인 화면 | 앱을 열면 노란색 로그인 화면이 나옵니다 |

| 3 | 이메일 + 비밀번호 입력 | 아무 이메일과 비밀번호(6자 이상) 입력 |
| 4 | 가입 완료 | 자동으로 메인 화면(대시보드)으로 이동 |

## 2단계: 현장 등록 / 第 2 步：添加工地

| 순서 | 화면 | 설명 |

| 3 | 주소 입력 | (선택사항) |
| 4 | "Add Site" 클릭 | 현장이 등록됩니다 |

---

## 3단계: 퇴근 체크인 (가장 중요!) / 第 3 步：下班打卡（最重要！）

**언제?** 매일 퇴근 전 (오후 5시 경)  
**누가?** 각 공종 반장 (목공, 전기, 설비, 도배...)  
**왜?** 앞 공정이 지연되면 뒷 공정에 자동 알림

### 방법 / 方法

| 순서 | 화면 | 설명 |
|------|------|------|
| 1 | 하단 "출퇴근" 탭 클릭 | 세 번째 탭 (체크 아이콘) |
| 2 | 현장 선택 | 작업 중인 현장 선택 |
| 3 | 공종 선택 | e.g. 목공, 전기, 설비, 도배 |
| 4 | 상태 선택 | 세 가지 중 하나 선택: |

**상태 3종 / 三种状态:**

```
✅ 완료 (Done/完工)
   → 오늘 작업 정상 완료

🕐 반일 연기 (Delayed Half/延误半天)
   → 반나절 정도 지연됨

🔴 하루 연기 (Delayed Full/延误一天)
   → 내일까지 작업 불가능
```

| 순서 | 화면 | 설명 |
|------|------|------|
| 5 | 사진 촬영 | 현장 사진 찍기 (필수) |
| 6 | 메모 (선택) | 특이사항 있으면 입력 |
| 7 | "Submit Check-in" 클릭 | 완료! |

**⚠️ 중요: "연기"를 선택하면 뒷 공정 반장에게 자동 알림이 갑니다!**

---

## 4단계: 현장 상태 확인 (대시보드) / 第 4 步：查看工地状态（看板）

| 순서 | 화면 | 설명 |
|------|------|------|
| 1 | 하단 "현황" 탭 클릭 | 첫 번째 탭 (대시보드 아이콘) |
| 2 | 신호등 확인 | 🔴 빨강 = 문제 있음 / 🟡 노랑 = 진행중 / 🟢 초록 = 모두 완료 |
| 3 | 공종별 상태 | 각 공종의 상태를 한눈에 확인 |

---

## 5단계: 문제 신고 (사진 + 음성 번역) / 第 5 步：问题上报（拍照 + 语音翻译）

**언제?** 현장에서 문제 발견 시 (누수, 크랙, 평활도 불량 등)  
**누가?** 작업자 누구나 (중국/베트남 노동자도 OK!)

### 방법 / 方法

| 순서 | 화면 | 설명 |
|------|------|------|
| 1 | 하단 "보고" 탭 클릭 | 두 번째 탭 (카메라 아이콘) |
| 2 | 사진 촬영 | 문제 부위 사진 찍기 |
| 3 | 사진에 표시 | "Mark on photo" 버튼 → 문제 부위에 동그라미 그리기 |
| 4 | 문제 유형 선택 | 태그 중 하나 선택: |

**문제 태그 / 问题标签:**
```
💧 Leak       누수 (漏水)
📏 Uneven     평활도 불량 (不平整)
🕳️ Crack      크랙 (裂缝)
🚿 Waterproof 방수 문제 (防水)
🧹 Dirty      청소 불량 (不洁净)
📌 Other      기타 (其他)
```

| 순서 | 화면 | 설명 |
|------|------|------|
| 5 | 음성 설명 | "Hold to speak" 길게 누르고 중국어로 설명 |
| 6 | AI 번역 | "Translate to Korean" 클릭 → 자동으로 한국어로 번역 |
| 7 | 전송 | "Send Report" 클릭 → 현장소장에게 전달 |

## 📊 신호등 색상 의미 / 红绿灯颜色含义

| 색상 | 의미 | 조치 |
|------|------|------|
| 🔴 빨강 | 하나 이상의 공종이 **지연** | 즉시 확인, 뒷공정 일정 조정 |
| 🟡 노랑 | 공사 **진행 중**, 이상 없음 | 정상 |
| 🟢 초록 | 모든 공종 **완료** | 다음 단계 준비 |
| ⚪ 회색 | 아직 데이터 없음 | 체크인 필요 |

## ❓ 자주 묻는 질문 / 常见问题

**Q: 앱을 설치해야 하나요?**  
A: 아니요. 카카오톡이나 브라우저에서 링크만 열면 됩니다. 설치 불필요.

**Q: 인터넷이 안 되면?**  
A: Wi-Fi 또는 데이터 연결이 필요합니다. 오프라인 모드는 추후 지원 예정.

**Q: 외국인 노동자가 사용할 수 있나요?**  
A: 네. 음성 번역 기능으로 중국어→한국어 자동 변환됩니다. 베트남어도 곧 지원 예정.

**Q: 사진은 어디에 저장되나요?**  
A: AWS 한국(서울) 리전에 안전하게 저장됩니다.
